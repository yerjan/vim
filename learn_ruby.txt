Find:
	there are general 5 finder:
	- find/detect 		-- return single object or nil
	- find_all/select
	any?
	all?
	delete_if

	$> (1..10).find {|i| i%3==0}
	$> 3

	$> (1..10).find_all {|i| i%3==0}
	$> [3, 6, 9]

	$> (1..10).any? {|i| i%3==0}
	$> true

	$> (1..10).all? {|i| i%3==0}
	$> false

	// notice it gets array!
	$> [1..10].delete_if? {|i| i%3==0}
	$> [1,2,4,5,7,8,9]

Merge:
	h1 = {"a"=> 111, "b"=>222}
	h2 = {"b"=> 333, "c"=>444}

	h2.merge(h1)
	{"a"=> 333, "b"=>333, "c"=>444}
	
	h1.merge(h2) {|key, old, new| new}
	{"a"=> 333, "b"=>333, "c"=>444}

	h1.merge(h2) {|key, old, new| old*5}
	{"a"=> 333, "b"=>1110, "c"=>444}

	// notice here h2 value is changed with exclamation mark.
	h2.merge!(h1)
